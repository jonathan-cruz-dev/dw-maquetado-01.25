Que es

HTML (HyperText Markup Language) es el lenguaje estándar utilizado para crear y estructurar páginas web. Es la base fundamental de la web y funciona junto con CSS (para el diseño) y JavaScript (para la interactividad).

Características

Definimos algunas Características:

  • Lenguaje de marcado: Usa etiquetas () para definir elementos como títulos, párrafos, imágenes, enlaces, etc.
  • Estructura jerárquica: Los elementos se anidan dentro de otros, formando un árbol de contenido (DOM).
  • No es un lenguaje de programación: No tiene lógica compleja como bucles o condicionales, solo define contenido.
