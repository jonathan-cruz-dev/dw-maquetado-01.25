¿Qué es CSS?

CSS es un lenguaje que define cómo se deben mostrar los elementos HTML en la pantalla. Controla aspectos como:

  • Colores
  • Fuentes
  • Tamaños
  • Posicionamientos
  • Animaciones y transiciones

Sinaxis Básica

Ejemplo

                h1 {
                    color: blue;
                    font-size: 36px;
                }

forma de Aplicar CSS

overflow

Es cuando el contenido se desorda del contenedor

En CSS, la propiedad overflow se utiliza para controlar qué sucede cuando el contenido de un elemento se desborda dentro de su contenedor. Cuando el contenido dentro de un elemento HTML es más grande que el área asignada (por ejemplo, si le das un height: 200px pero el contenido mide más), se produce un desbordamiento (overflow).
