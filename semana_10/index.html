¿Qué es CSS?

CSS es un lenguaje que define cómo se deben mostrar los elementos HTML en la pantalla. Controla aspectos como:

  • Colores
  • Fuentes
  • Tamaños
  • Posicionamientos
  • Animaciones y transiciones
